# Learning Projects

Welcome to **Learning-Projects**! 🚀  
This repository is a personal collection of project-based explorations covering the core concepts and best practices of **React** and related front-end technologies. It is ideal for beginners and intermediate developers who wish to strengthen their hands-on React skills by building small, meaningful applications.

---

## 📁 Project Structure

Below is a list of the primary projects and learning modules included in this repository:

| Project                          | Description                                                                                | Technologies       |
|-----------------------------------|--------------------------------------------------------------------------------------------|--------------------|
| **1st-Learning-Project**          | Starting point for React fundamentals: setup with Vite, simple component structure, basics. | React, Vite        |
| **02Color-Changer**               | Interactive app to dynamically change background colors using buttons.                      | React, Vite        |
| **03Password-Generator**          | Customizable password generator with adjustable length/features.                            | React, Vite        |
| **04CurrencyConvertor**           | Real-time currency converter consuming public APIs for exchange rates.                      | React, Vite, APIs  |
| **AdvancedTask_Manager**          | A feature-rich task manager demonstrating advanced state management and UI updates.         | React, Vite        |
| **Custom-React-learning**         | Pure JS custom React implementation for educational purposes.                              | JavaScript, HTML   |

---

## ✨ Highlights

- **Modular Learning:** Each folder contains a standalone mini-project, focusing on a single concept or a small set of related concepts.
- **Modern Tooling:** Most projects are initialized with Vite for fast development and hot module replacement.
- **Best Practices:** Sample usage of hooks (useState, useEffect), component composition, lifting state, and more.
- **API Integration:** Demonstrates how to interact with 3rd-party APIs (see Currency Converter).
- **From Basics to Advanced:** Starts from simple UI changes and progresses to real-world apps like a password manager and a task tracker.

## 🛠️ Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/arin-gupta06/Learning-Projects.git
   cd Learning-Projects
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run any project**
   - Navigate to the project folder of your choice (most use Vite).
   - Start the dev server:
     ```bash
     cd 02Color-Changer
     npm install
     npm run dev
     ```
   - Open the provided local URL in your browser.

**Note:** Some folders may require their own node_modules due to separate project structures.

## 📦 Project List Overview

### 1st-Learning-Project
- Foundation for learning React, project setup using Vite.

### 02Color-Changer
- Click a button to instantly change the background color.  
- Teaches useState and basic event handling.

### 03Password-Generator
- Generate strong passwords of variable lengths with character-type selection.

### 04CurrencyConvertor
- Enter an amount and choose currencies to convert in real-time using exchange rate APIs.

### AdvancedTask_Manager
- Manage daily tasks with features like add, delete, and mark as completed.

### Custom-React-learning
- Explore how React works under the hood with a custom*JS implementation of a mini React-like system.
